Este repositorio contiene mi **Currículum Vitae interactivo**, desarrollado con **Angular** y diseñado con una estética gamer (o eso se intentó).  
La idea es mostrar mis habilidades técnicas y creativas en un formato dinámico y accesible.

## Tecnologías utilizadas
- Angular (TypeScript, HTML, CSS)
- UI/UX con enfoque visual gamer
- Modelado de datos 

## Objetivo
Presentar mi perfil profesional de manera atractiva y accesible, integrando skills en un entorno web que cualquiera pueda explorar fácilmente.
